7. In de code staat iets ivm met favicon <link rel="icon" href="***.png" />
Zoek op het internet wat dit stukje code betekent.
    Het waarde bij link rel="icon" importeert een icoontje voor de website. Dit kan zowel een favicon.ico of een afbeelding zijn. In dit geval is deze afbeelding een PNG, href="***.png" zoekt naar de bron PNG afbeelding in de folder.
    
Waar zie je het effect van deze lijn code in Chrome?
    Je ziet linksboven in de tabladen het gegeven afbeelding/icoontje van de website. Als dit niet gegeven is het icoontje meestal een stock icoontje.
